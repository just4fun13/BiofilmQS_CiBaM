# BiofilmQS_CiBaM
Hybrid cellular automaton-based model for quorum sensing-controlled biofilm evolution

This repository contains the source code and executable build used in the manuscript.

How to run:
1. Download the Windows build from Releases.
2. Unzip the archive.
3. Run BiofilmQS.exe.
4. Load preset of parameters stored in configs/default_config.json by clicking "Load preset" button
5. Run simulation.
